## Todo App for React

React（TypeScript）を使った、簡単な Todo アプリ開発

### Start

```shell
# node バージョン確認
node -v

# package install
npm install

# server start
npm run start:legacy
```

### React プロジェクト開始方法

```shell
# JS
npx create-react-app { projectName }
# TS
npx create-react-app { projectName } --template typescript

# サーバー起動
cd { projectName }
npm start
```
